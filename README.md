# sit323-737-2023-t1-prac5d
Two node.JS applications inside a docker image networked together.

## Installation Instructions

### 1 - Change the Directory
```
cd docker-network
```

### 2 - Build and Deploy Docker Image
```
docker-compose up -d build
```

### 3 - Access Webpages
App1:
```
http://localhost:3000
```
App2:
```
http://localhost:4000
```
